# AWS WAFv2 Fail2ban Integration

AWS WAFv2とFail2banを統合したセキュリティソリューション

## 概要

このプロジェクトは、AWS WAFv2とFail2banを組み合わせて、Webアプリケーションのセキュリティを強化するためのソリューションです。

## 機能

- 自動的な攻撃検知とブロック
- AWS WAFv2ルールの動的更新
- ログ監視とアラート機能

## セキュリティ要件

- OWASP準拠のセキュリティ設定
- 認証情報の暗号化
- Infrastructure as Codeによる安全な構成管理
